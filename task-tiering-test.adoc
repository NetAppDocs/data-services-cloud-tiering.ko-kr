NetApp Cloud Tiering에서 데이터 계층화를 설정하기 전과 후에 ONTAP 클러스터에서 개체 저장소로의 네트워크 지연 시간과 처리량 성능을 측정하기 위해 클라우드 성능 테스트를 실행합니다.  이 테스트를 통해 발생한 모든 실패도 식별됩니다.
